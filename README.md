Personal Resume Website

This is my personal resume website built using **HTML and CSS**.
It showcases my profile, skills, projects, and contact information in a clean and responsive layout.

Features
* Simple and clean UI design
* Fully responsive layout
* Organized sections (About, Skills, Projects, Contact)
* Lightweight and fast loading

🛠️ Tech Stack
* HTML5
* CSS3

Project Structure

* `index.html` – Main webpage
* `style.css` – Styling file
* `assets/` – Images and other resources


Live Demo
https://jairamhegde.github.io/resume/

Purpose
This project is created to:
* Practice frontend fundamentals
* Build a personal portfolio
* Showcase my work in a simple format
